# Singly Linked List in Java 

A clean and simple implementation of a Singly Linked List data structure from scratch using Java. This project demonstrates core Data Structures and Algorithms (DSA) concepts such as node manipulation, reference passing, and dynamic memory allocation.

## Features

* **Insert at End:** Add a new node to the tail of the list.
* **Insert at Start:** Add a new node to the head of the list.
* **Delete Node:** Remove a specific element from the list by value (safely handles head, middle, and end deletions by bypassing references).
* **Find / Search:** Traverse the list to check if a specific value exists.
* **Display List:** Print the entire linked list in a visually clear format (e.g., `10 -> 20 -> 30 -> Null`).

## Technologies Used

* **Java** (Core Java / JDK 8+)

## Core Concepts Covered

* Object-Oriented Programming (OOP)
* Memory Reference Passing in Java (Instead of pointers)
* Data Structure Traversal using `while` loops
* Edge case handling (Empty lists, missing elements)
